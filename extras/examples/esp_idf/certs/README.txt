Certificates generated by https://github.com/SUPLA/supla-esp-signtool
Don't use files from this folder for production purposed. Those are only
provided as an example. Generate separate keys for each application/board
and keep your private key in a secure location.

I.e.:
Generate public and private key:
supla-esp-signtool -k certificate_file_name -g

Generate .c file based on public key:
supla-esp-signtool -k certificate_file_name.pub -c rsa_public_key.h

Sign app:
supla-esp-signtool -k certificate_file_name -s esp_application.bin
